Hazard Model

def sigmoid(x):
    return 1.0 / (1.0 + np.exp(-x))

def predict_job_loss_prob(params, u, ind_risk, r):
    x = params.beta0 + params.beta_unemployment * u
         + params.beta_industry * ind_risk
         + params.beta_interest_rate * r
    return float(sigmoid(np.array([x]))[0])

Optimal Hedge

def optimal_hedge_ratio(losses, hedge_payoffs):
    """h* = Cov(Loss, Hedge) / Var(Hedge)"""
    cov = np.cov(losses, hedge_payoffs)[0, 1]
    var = np.var(hedge_payoffs)
    return cov / var if var > 1e-10 else 0

FAQs

What is this?

We help you hedge your income against job loss using prediction markets. Enter your profile (industry, salary, etc.) and we run a Monte Carlo simulation to recommend how many Kalshi unemployment contracts to buy.

How does the hedge work?

You buy contracts that pay out if US unemployment exceeds a threshold (e.g., 8%). If you lose your job when unemployment is high, the payout helps cover lost income. We size the hedge to match ~6 months of pay.

What markets does it use?

We use Kalshi’s unemployment and recession markets. These are regulated prediction markets where you can trade on economic outcomes.

Is this financial advice?

No. This is a research tool from NYU BnF. Always do your own research and consider consulting a financial advisor before making investment decisions.
