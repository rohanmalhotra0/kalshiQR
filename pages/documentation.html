Overview of how we built the job-loss hedging model

What we built

A Monte Carlo framework for hedging personal income risk using prediction market contracts (Kalshi). The model simulates job-loss events via a macro-sensitive hazard process and evaluates how unemployment-linked contracts reduce tail risk in income distributions.

Household income risk is usually addressed through savings or insurance, but these offer limited protection against macroeconomic shocks. Prediction markets like Kalshi offer tradable contracts tied to unemployment, recessions, and interest rates—giving individuals a way to hedge job-loss risk.

System pipeline

We built a pipeline that flows from personal inputs through macro data, hazard modeling, stochastic simulation, and finally to hedge recommendations:

personal data
↓
macro data (FRED/BLS)
↓
hazard model
↓
job-loss probability (λ)
↓
Poisson stochastic model
↓
Monte Carlo simulation
↓
income distribution
↓
Kalshi market data
↓
optimal hedge ratio (h*)
↓
hedge portfolio

Model design

Income process: Employment state E_t ∈ {0,1}. Cumulative wealth dW_t = S·E_t·dt where S is salary.

Job loss & reemployment: Poisson job loss with intensity λ_t. Employment flips 1→0 on job loss. Unemployment duration T ~ Exponential(μ), μ = 20 weeks (BLS). After T weeks, employment flips 0→1 (reemployment). So income path is: employed → unemployed → employed.

Hazard rate: P(JobLoss) = σ(β₀ + β₁·U_t + β₂·industry + β₃·r_t) — logistic function of unemployment, industry risk, and interest rate.

Hedge: Kalshi contracts are binary ($1 if event occurs, $0 otherwise). Optimal hedge ratio h* = Cov(Loss, Hedge) / Var(Hedge). Hedge threshold trades off: lower (e.g. 5.5–6%) = more correlation with layoffs but higher payout frequency; higher (e.g. 8%) = rarer trigger, weaker correlation.
